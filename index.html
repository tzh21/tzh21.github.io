点云重建笔记

Posted on 2024-02-05 Edited on 2024-02-06

SDF

SDF 全称 Signed Distance Function，是一个函数 \(\mathbb{R}^{3} \to \mathbb{R}\) 。一个物体唯一对应一个 SDF。SDF 输入为三维空间中点的坐标，输出为到该物体表面的距离（正负号表示内外）。

已知一个物体的 SDF，则可完全掌握物体的表面信息。一个物体的 SDF 的零点即该物体的表面。

点云重建的思路是通过有限点对应的 SDF 值拟合 SDF，拟合 SDF 的零点即为物体表面的近似。

DeepSDF

借助 latent vector，可以让一个神经网络建模多个 SDF。设神经网络为 \(f_{\theta}\)，第 \(i\) 个形状对应 latent vector \(z_{i}\)，取空间中的点为 \(x\)，目标是训练神经网络使得 \(f_{\theta} (z_{\theta},x) = SDF(x)\)。重点在于如何得到 latent vector \(z_{i}\)。

作者使用 auto decoder 来获取 latent vector。获取 latent vector 的一个方法是使用自编码器（auto encoder）。但作者认为这种方法需要多训练一个网络，浪费了资源。作者提出用 auto decoder 解决。即将一个神经网络分成两层，第一层预测 latent vector，第二层用得到的 latent vector 和位置坐标预测 SDF。

OccNet

以往的 3D 图形表示方法大致分为三类：voxel, point, mesh（网格）。不能兼顾内存效率和生成效率。voxel 的缺点在于占用空间随分辨率增长速度过快。point 缺点在于需要进一步处理才能得到几何图形。mesh 缺点在于依赖模板，因此不允许表示任意的图形。

occupancy function 是用来表达连续三维空间中任一点是否在物体内的函数。 occupancy function 是一个函数 \(\mathbb{R}^{3} \to \{ 0,1 \}\)，输入为空间中一点的坐标，输出为该点是否在物体内部。

occupancy network 是用来表示连续三维空间中任一点在物体内的概率的函数。occupancy network 是一个函数 \(\mathbb{R}^{3} \times \chi \to [0,1]\)，输入为所求点的坐标（\(\mathbb{R}^{3}\)）和训练网络用的样本（\(\chi\)），输出为所求点在物体内部的概率。

损失函数使用二分类问题中常见的交叉熵函数，取样方法使用均匀分布。
